# 引用语法
  引用相当于是变量的别名共享地址，操作引用就是操作本身所以：
  ```cpp
  swap(inta,intb);//别处的调用
  void swap(int &a,int &b){
    int tmp=a;
    a=b;
    b=tmp;
  }
  这个代码可以正常交换。

  左值引用：
  ```cpp
  int &a=b;

  常量引用：
   ```cpp
  const int &a=b;
  不可以通过a修改b

  函数传递和返回值：
  直接的值传递会建立副本，所以用引用传递本身。但是注意作为一个返回值的时候需要static注意作用域
  ```cpp
  int & a(){
    return b;
  }

  a()=200; //修改返回值,虽然不觉得有什么用
